# API Integration Project - Education Data Portal

## Objective
Demonstrate REST API integration, JSON parsing, and structured data transformation using the Urban Institute Education Data API.

## Skills Demonstrated
- REST API calls using Python `requests`
- Handling HTTP status codes
- Query parameters & filtering
- JSON parsing
- Converting API responses into Pandas DataFrames
- Basic enrollment analysis

## Endpoint Example
https://educationdata.urban.org/api/v1/schools/ccd/enrollment/2013/grade-3/?charter=1&fips=11
